도커는 리눅스의 격리 기술을 이용하여 컨테이너를 생성해 가상화 및 프로세스 관리를 진행할 수 있게 해 주는 유용한 기술이다.

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
